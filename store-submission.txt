--- NAME ---
IA Helper

--- SHORT DESCRIPTION (<=132 chars) ---
Multi-provider AI assistant (Ollama local, OpenAI, Anthropic, Groq) - context menu, translation, correction, chat, agents.

--- LONG DESCRIPTION ---
IA Helper is a browser extension that brings a powerful AI assistant into your browser.

MAIN FEATURES:
- Smart context menu with contextual actions
- Multi-provider support: Ollama (local), OpenAI, Anthropic, Groq, OpenRouter
- Instant translation (12 languages)
- Correction, rephrase, summarize, explain
- 24 professional presets with 140+ specialized actions
- Streaming responses into input fields
- Interface in 5 languages (EN, FR, ES, IT, PT)

PRIVACY:
- No personal data collection
- No tracking or ads
- Works 100% locally with Ollama if you choose
- Text only goes to the provider you configure

USAGE:
1. Select text on any page
2. Right-click > IA Helper > choose an action
3. Response appears instantly
