Demarrage rapide

IA Helper est un assistant IA intelligent pour Chrome connecte a un serveur Ollama. Il vous permet d'utiliser des modeles d'IA locaux pour analyser, corriger, traduire et reformuler du texte.

Prerequis

  • Un serveur Ollama installe et en cours d'execution
  • Au moins un modele telecharge (ex: llama3, mistral, gemma)
  • Google Chrome (version 88+)

Installation

  1. Ouvrez chrome://extensions/
  2. Activez le "Mode developpeur"
  3. Cliquez sur "Charger l'extension non empaquetee"
  4. Selectionnez le dossier ia-helper

Configuration

Accedez aux options via clic droit > IA Helper > Options ou en cliquant sur l'icone du plugin.

Connexion au serveur

  • URL du serveur: Par defaut http://localhost:11434
  • Modele: Selectionnez le modele a utiliser
  • Streaming: Activez pour voir les reponses en temps reel

Langues

  • Langue interface: FR, EN, ES, IT, PT
  • Langue reponse IA: Force l'IA a repondre dans une langue specifique

Menus contextuels

IA Helper ajoute des options au menu contextuel (clic droit) selon le contexte:

Sur du texte selectionne

  • Resumer le texte
  • Traduire (vers plusieurs langues)
  • Expliquer simplement
  • Extraire les points cles
  • Analyser le sentiment

Dans un champ de saisie

  • Corriger les erreurs
  • Traduire
  • Reformuler en Mail Pro
  • Developper le contenu
  • Ameliorer le style

Sur une page (sans selection)

  • Resumer cette page
  • Extraire les points essentiels
  • Traduire cette page

Presets Professionnels

Activez des ensembles d'actions specialisees pour votre metier:

  • Support IT: Resume ticket, suggerer solution, rediger reponse
  • Service Client: Analyser sentiment, reponse empathique
  • Developpeur: Expliquer code, revue de code, debug
  • Etudiant: Expliquer simplement, creer resume, questions
  • Redacteur: Ameliorer style, optimiser SEO, idees titres
  • Commercial: Analyser prospect, argumentaire, email suivi
  • Chercheur: Points cles, identifier biais, resume article

Actions personnalisees

Creez vos propres actions avec des prompts personnalises:

  1. Allez dans Options > Personnalisation
  2. Cliquez sur "Ajouter une action"
  3. Donnez un nom et choisissez le contexte
  4. Redigez le prompt systeme
  5. Vos actions apparaitront dans "Mes actions"

Mode Streaming

Le streaming permet de voir la reponse de l'IA en temps reel.

  • Dans un champ de saisie: le texte s'affiche directement
  • En analyse: le resultat s'affiche progressivement

Support multilingue

IA Helper supporte 5 langues: Francais, English, Espanol, Italiano, Portugues.

Vous pouvez forcer l'IA a repondre dans une langue specifique (sauf pour les traductions).

Depannage

L'extension ne fonctionne pas

  • Verifiez que Ollama est en cours d'execution
  • Testez la connexion dans les options
  • Rechargez l'extension dans chrome://extensions

Erreur de connexion

  • Verifiez l'URL du serveur (par defaut: http://localhost:11434)
  • Assurez-vous qu'aucun pare-feu ne bloque la connexion
